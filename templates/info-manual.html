FAQ section


If you're unable to find the answer here, feel free to reach out to us!
Unable to mark attendace
In order to mark attendace, make sure you check the right checkbox present in front of the student name/rollno.
Issue with view attendance
At times due toserver lag, we might not fetch the result quickly. Be patient you'll se the results. If not feel free to reach out to us.
Class entry not found in mark attendance page
if the entry is not found in mark attendace page, make sure you have the uthorizartion to manipulate the respectieve class entries. If not, you can always go ahed and create class entries in add student section.
Google Drive not shared with student
In order to share google drive with class, you will have to configure the sharable resourses in the profile section.
Library return not availble
Once the deadline to return books is passed you are expected to come down to the office an file a return. You will be asked to pay the fine an most cases for the same.
Time Table not updtes
Time tables will be updated by admin, once the same is done you will se the changes!
